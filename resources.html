Demo Application and Source Code ​

The OpenSCD web application is on openscd.github.io. The source code repositories can be found under the GitHub organization github.com/openscd.

Contact and Community ​

Discussions are on the ZULIP chat: openscd.zulipchat.com.

The OpenSCD wiki is here: github.com/openscd/open-scd/wiki.

Reporting Issues and Contributing ​

For bug reports and feature requests, please use the GitHub issue tracker.

Other Projects Using OpenSCD ​

CoMPAS is a project about IEC 61850 engineering, hosted by LF Energy. The main driver is Alliander.
OpenSCD is used as a graphical front-end there.

OpenEnergyTools is a test-bed repository with a fork of OpenSCD including plugin prototypes.
While its maintainer Jakob Vogelsang does not recommend it for productive use, it serves as a valuable example for developers and can be used to preview upcoming features.
